# J.A.R.V.I.S. Mobile Terminal

A functional, sleek mobile terminal interface built with React Native and Expo.

## Features
- **Real-time Command Simulation**: Interactive terminal prompt.
- **Custom Commands**: Includes `help`, `status`, `whoami`, `scan`, and `clear`.
- **Design**: Minimalist high-tech aesthetic with neon cyan and green accents.

## How to Run
1. Install Expo Go on your mobile device.
2. Clone this repository: `git clone https://github.com/davidbumb87-dot/jarvis-mobile-terminal.git`
3. Install dependencies: `npm install`
4. Start the server: `npx expo start`
5. Scan the QR code with your phone to launch the terminal.
